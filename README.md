# Task-Manager

## Tech Stack

- Javascript
- Mongoose
- HTML/CSS
- ES2017
- NodeJS

## Project setup / Installation

For installing the dependencies required by the VueJs project

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```
